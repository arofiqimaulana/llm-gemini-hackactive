# llm-gemini-hackative
Merupakan course dari Hackactive mengenai LLM menggunakan gemini. Terdiri dari beberapa section yaitu
1. Pengenalan Course
2. Introduction Generative AI
    - Definisi AI, Generative AI, LLM
    - Generative vs Discriminative Models
    - Transformers dan Diffusion Models
    - Kegunaan HuggingFace
3. Generative AI using Gemini
    - Prompting Technique
        - CoT - Chain-of-Thought
        - ToT - Tree of Thoughts
        - ReAct
    - Parameter 
        - max_output_tokens
        - temperature
        - top_k
        - top_p
    - Function Calling
4. RAG
    - Vector Database
        - Embedding Model
        - Indexing Mechanis
        - Similarity Metrics
        - Storage Engine
    - LangChain
        - Model
        - Prompt
        - Chain
        - Memory
5. Streamlit
    - AI Agent vs AI Assistant
    - Build Chatbot
